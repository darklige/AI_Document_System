# **《GB 47372-2026 移动电源安全技术规范》全面解析**

本文对中华人民共和国国家标准 **《移动电源安全技术规范》（GB 47372-2026）** 进行逐章、逐条的系统性解读与结构化梳理，涵盖其适用范围、核心术语、试验条件、安全要求、材料规范、生产与运输要求以及附录中的关键技术细节。该标准将于 **2027年4月1日起正式实施**，是对移动电源（俗称“充电宝”和“户外电源”）产品安全性提出的强制性、全方位的技术框架。

---

## **一、基本信息**

- **标准编号**：GB 47372-2026  
- **中文名称**：移动电源安全技术规范  
- **英文名称**：Safety technical specification for power bank  
- **发布日期**：2026年3月31日  
- **实施日期**：2027年4月1日  
- **提出与归口单位**：中华人民共和国工业和信息化部  
- **标准性质**：**强制性国家标准（GB）**  

> 💡 **意义**：此标准填补了此前在移动电源整机层面缺乏统一强制安全规范的空白，将极大提升消费者使用安全性，推动行业规范化发展。

---

## **二、适用范围（第1章）**

### ✅ 适用产品：
本标准适用于：
- **额定输入电压为交流220V** 或 **直流不大于250V**；
- 输出为 **直流或交流** 的移动电源。

> 🔍 注：交流220V包含宽电压输入范围（如 AC 100V～240V）。

### ❌ 不适用产品：
- 具备反向充电功能的手机、平板、笔记本电脑等设备，即使能给其他设备供电，也不属于本标准定义的“移动电源”。

## **四、术语和定义（第3章）**

本章明确了移动电源及相关组件的关键概念，确保理解一致。

| 术语 | 定义要点 |
|------|---------|
| **3.1 移动电源 (Power Bank)** | 可携带、由电池/电池组+电路+外壳组成，提供稳定直流/交流输出，质量 ≤18kg 的电源系统。分为两类： |
| **3.2 便携式移动电源** | 总额定能量 ≤ **160Wh**（常见“充电宝”） |
| **3.3 便携式储能电源** | 总额定能量 > **160Wh**（俗称“户外电源”） |
| **3.9 额定能量 (Rated Energy)** | 电池/电池组总能量（单位：Wh），计算方式：<br> `∑(额定容量 × 标称电压)`，结果向上取整 |
| **3.10 漏液 (Leakage)** | 非设计性的可见液体电解质泄漏 |
| **3.11 起火 (Fire)** | 出现可见火焰（火花不算） |
| **3.12 爆炸 (Explosion)** | 外壳剧烈破裂且主要部件抛射出去 |
| **3.24 热失控 (Thermal Runaway)** | 因放热反应引发电池不可控温升的现象（重大安全隐患） |
| **3.25 磁性异物 (Magnetic Impurity)** | 材料中可被磁铁吸附的铁、铬、镍、锌等杂质（易导致内部短路） |

## **五、试验条件（第4章）**

### 4.1 试验环境条件
除非另有规定，所有试验应在以下条件下进行：
- 温度：25℃ ± 5℃  
- 相对湿度：≤ 75%  
- 大气压力：86 kPa ~ 106 kPa

### 4.2～4.3 充电/放电程序
- 按制造商规定的方法进行充/放电。
- “充满电”、“放完电”状态应在产品本体或说明书中有明确指示。

### 4.4 预处理
所有样品需先完成 **2个完整的充放电循环**，每个循环后静置 ≥1h。

> 🔋 锂离子电池按 GB 31241 执行；其他类型电池可参考 GB 31241 或相应标准。

### 4.5 单一故障条件（Single Fault Condition）
- 模拟故障应**逐项施加**，一次只模拟一个元器件故障。
- 典型故障包括：
  - 半导体器件（如MOSFET）任意两脚短路/开路
  - 保险丝短路/开路
  - 电容短路/开路
  - 限压器件（如TVS）短路/开路

> ⚠️ 已符合 GB 4943.1-2022 附录G要求的安全元器件，无需再进行单一故障测试。

### 4.6 型式试验

#### 4.6.1 样品要求
- 必须是代表最终用户的**量产代表性样品**（不可使用梯次利用电池）。
- 试验导线总电阻 < 20 mΩ。
- 样品生产时间应在 **8个月以内**。
- 所有样品必须完成 **4.4 预处理** 后方可进行型式试验。

## **六、基本要求（第5章）**

### 5.1 一般要求
- 整机安全应满足本文件及 **GB 4943.1** 的要求。
- 所用电池/电池组应满足相应电池标准（如 GB 31241）。
- 锂离子电池编码必须符合 **GB/T 45565**。
- 若无对应国家标准，可参考 GB 31241 和 GB/T 45565。

### 5.2 标识和警示说明

#### 5.2.2 警示说明（中文）
必须包含以下内容：
- 摔落、磕碰可能存在安全风险；
- 切勿置于高温环境中；
- 若出现严重鼓胀，请勿继续使用；
- 禁止拆解、挤压或投入火中。

#### 5.2.3 耐久性测试
- 经受 **水擦15秒 + 75%酒精擦15秒** 后，标识仍应清晰可辨，铭牌不得卷边脱落。

### 5.3 外壳阻燃
外壳材料必须满足以下任一条件：
- 符合 **UL94 V-0** 级阻燃等级；
- 符合 **GB 4943.1-2022 中 S.1 和 S.2** 的防火要求。

### 5.4 交流输入/输出连接装置
- 若涉及市电（220V AC），其插头/插座/耦合器必须符合：
  - GB 1002 / GB/T 1003（家用插头插座）
  - GB/T 2099.1（通用要求）
  - GB/T 11918 / GB/T 17465（工业/器具耦合器）

## **七、移动电源安全要求（第6章）**

### 6.1 充电电压控制
- 正常条件下，施加 **1.2×n×Uup** 电压，应触发过压保护，**单节电压不超 Uup**；
- 单一故障条件下，也应能有效保护，**总电压不超 n×Uup**；
- 可绕过 DC-DC 模块，在保护电路输入端测试。

> ✅ n = 电池串联级数；Uup = 单节充电上限电压

### 6.2 充电电流控制
- 在正常与单一故障下，实测最大充电电流 ≤ **Icm**（最大充电电流）或 **m×Icm**（并联时）；
- 不考虑 **3秒内** 的瞬态峰值。

> ✅ 仅适用于保护电路在主板上的产品

### 6.3 放电电压控制
- 放电最低电压 ≥ **Udo**（放电截止电压）；
- 不考虑 **1秒内** 的瞬态低谷。

### 6.4 放电电流控制
- 最大放电电流 ≤ **Idm** 或 **m×Idm**；
- 不考虑 **3秒内** 的瞬态高峰。

### 6.5 充放电温度控制
当温度超出规定范围时，**必须停止充放电**。

| 测试场景 | 设定温度 | 要求 |
|----------|----------|------|
| 充电高温保护 | Tcm + 4℃ | 达到前应停止充电 |
| 充电低温保护 | Tci - 4℃ | 达到前应停止充电 |
| 放电高温保护 | Tdm + 4℃ | 达到前应停止放电 |

> ✅ Tcm/Tci/Tdm 为电池参数（非电池组）

### 6.6 端口短路保护
- 所有输入/输出端口施加 **(80±20)mΩ** 短路电阻，持续 **2小时**；
- 电池最热点温升 ≤ **10K**；
- 无起火、爆炸、漏液。

> ✅ 可选择最不利端口或多端口同时短路

### 6.7 端口误操作
- 输出口被外来电源反向充电1小时（模拟误插）；
- 输出口与输入口用线缆直连1小时；
- 均不得起火、爆炸、漏液。

> ✅ 交流端口除外

### 6.8 智能管理

#### 功能要求：
1. **状态监测**
   - 单节电池电压（误差 ±1%）
   - 电池温度（≤70℃ 时 ±3℃；＞70℃ 时 ±6℃）

2. **异常信息存储**
   - 过充电压（超过 Uup 的记录）
   - 异常高温（超过 Tcm 或 Tdm）
   - 发生时间（精确到分钟）

3. **用户可读取**
   - 异常记录
   - 电池型号、编码（含生产厂、生产日期）

#### 通信方式：
- 显示屏、APP、PC软件等；
- 支持 USB-C PD 协议通信（见附录A）

> ✅ 存储期限 ≥ 建议安全使用年限

### 6.9 电压调节（防老化设计）

移动电源应具备动态降低最高充电电压的功能，以延长寿命、提高安全性。

#### 两种调节方式（任选其一）：

##### **表3：按时间调节**
| 使用时间（自生产日起） | 最高充电电压上限 |
|------------------------|------------------|
| ≤18个月 | ≤ n×(Uc1 - 0.1V) |
| ≤30个月 | ≤ n×(Uc1 - 0.15V) |
| ≤42个月 | ≤ n×(Uc1 - 0.2V) |

##### **表4：按循环次数调节**
| 满电循环次数 | 最高充电电压上限 |
|--------------|------------------|
| ≤70次 | ≤ n×(Uc1 - 0.1V) |
| ≤140次 | ≤ n×(Uc1 - 0.15V) |
| ≤210次 | ≤ n×(Uc1 - 0.2V) |

> ✅ Uc1 为电池充电限制电压；n 为串联级数

### 6.10 异常禁用功能

#### 必须具备：
- **欠压禁用**：任意单节电压降至 0.4×Udo → 禁止充放电；
- **过压禁用**：任意单节电压升至 Uup + 0.3V → 禁止充放电，且**不可恢复**（仅允许一次性短暂充电复位）。

> ✅ 测试后仍不能恢复正常充放电

## **八、电池安全要求（第7章）**

### 7.1 过充电
- 充满电后，以 **0.2It** 恒流充电至 **1.3×Uc1**，转恒压充1h 或总时间达7h；
- **不得起火、爆炸**。

### 7.2 挤压
- 充满电后，用直径25mm钢棒以 **0.1mm/s** 速度挤压；
- 压力达到 **20kN±1.2kN（硬壳）** 或按表5（软包）设定；
- 或电压下降1/3时停止；
- **不得起火、爆炸**。

#### 表5：软包电池挤压力
| 电池宽度 (mm) | 挤压力 (kN) |
|----------------|-------------|
| (0,25]         | 2           |
| (25,30]        | 6           |
| ...            | ...         |
| (>75)          | 20          |

### 7.3 热滥用
- 充满电后放入试验箱，以 **(5±2)℃/min** 升温至 **135℃±2℃**，恒温 **60分钟**；
- **不得起火、爆炸**。

### 7.4 针刺
- 充满电后，用 **Φ4mm 耐高温钢针**（锥角14°）以 **(20±1)mm/s** 穿刺电池中心；
- 钢针停留5分钟；
- **不得起火、爆炸**。

> ✅ 针对面可预留Φ12mm孔

### 7.5 析锂（锂沉积检测）

#### 目的：防止长期使用后析出金属锂，引发热失控。

#### 方法：
1. 循环条件：
   - 充电：0.8×Icm 至 (Uc1 - 0.1V)
   - 放电：1It 至 Ude
   - 间隔10分钟
2. 累计放电容量 ≥ **225C** 后拆解；
3. 若中途连续几次放电容量 < 额定容量，则提前拆解。
4. 拆解后检查负极：
   - ≥0.2mm² 的析锂点 ≤ **5个**
   - 总面积 ≤ **1mm²**
   - 图4所示豁免区域不计入

> ✅ 仅适用于锂离子电池

### 7.6 热失控（仅便携式储能电源）
- 按 **附录B** 进行加热触发试验；
- 加热功率依电池能量选择（见表B.1）；
- 判定条件：电压降 >5% 且 dT/dt ≥1℃/s 持续3s，或温度达厂商上限；
- **不得起火、爆炸**。

## **九、电池材料要求（第8章）**

### 8.1 总体要求
- 企业必须具备材料检测能力；
- 每批次材料来料及使用前必须检测；
- 电解液每周至少检测一次。

|------|-----------|---------------|------------|

| 锰酸锂 | ≤400 | ≤1000 | ≤40 |
| 磷酸铁锂 | ≤1000 | ≤1000 | ≤100 |
| 镍钴锰 | ≤400 | ≤150 | ≤30 |

| 类型 | 磁性异物 | Fe | Cu+Zn |
|------|--------|----|------|
| 石墨 | ≤1 ppm | ≤50 ppm | ≤40 ppm |
| 硅碳 | ≤1.5 ppm | ≤150 ppm | ≤40 ppm |
| 硬碳 | ≤3 ppm | ≤150 ppm | ≤40 ppm |

> ✅ 检测方法同上

### 8.4 隔膜材料（表7）
| 项目 | 指标 |
|------|------|

> ✅ 检测方法：拉伸/伸长→GB/T 36363；穿刺→附录E；热缩→附录F

### 8.5 电解液
| 项目 | 要求 |
|------|------|
| 水分含量 | ≤20 ppm |
| 氢氟酸（HF）含量 | ≤200 ppm（来料≤100 ppm） |

> ✅ 水分→附录G（库仑法卡尔·费休）；HF→附录H（电位滴定）或附录I（化学滴定）

## **十、生产与运输要求**

### 9 生产质量管理
- 锂离子电池生产应符合 **GB/T 47292.3**
- 电池组生产应符合 **GB/T 47292.4**
- 钠离子等新型电池可参照执行

### 10 运输安全要求
- 必须满足 **联合国《关于危险货物运输的建议书》试验和标准手册》第38.3节** 的要求；
- 航空携带限制：
  - ≤100Wh：可随身携带
  - 100～160Wh：需航空公司批准
  - ＞160Wh：禁止携带

## **十二、总结与影响**

### ✅ 本标准的核心亮点：
1. **首次建立整机级强制安全标准**，覆盖“整机+电池+材料”全链条；
2. **引入智能管理功能**，要求具备异常监测、记录与读取能力；
3. **强调生命周期安全管理**，通过“电压调节”和“异常禁用”防止老化风险；
4. **严控原材料品质**，对磁性异物、水分、HF等设限；
5. **强化极端工况测试**，如热失控、析锂、误操作等。

### 📈 对行业的影响：
- 提高准入门槛，淘汰低端劣质产品；
- 推动企业加强BMS研发、材料品控和数据追溯能力；
- 提升消费者信心，减少安全事故。
